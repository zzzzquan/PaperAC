# 情智文检——高情商AIGC（PaperAC）

> 域名: paparac（规划中）

情智文检（PaperAC）是面向课程论文场景的AIGC检测产品，支持上传PDF并生成检测报告。AIGC疑似结果按“高度疑似/轻度疑似”分级，比例固定为7:3，合计占比为用户输入x。

## 核心规则
- **分级比例:** 高度疑似/轻度疑似 = 7:3
- **总占比:** 高低疑似合计 = 用户输入x
- **算法方向:** 数学算法（占位，后续设计）

## 功能概览
- PDF上传与任务编排
- 正文抽取与句子拆分
- AIGC疑似分级与占比控制
- 总览与标红报告生成
- 报告下载与归档
- 账号与配额管理（占位）

## 使用流程（占位）
1. 注册/登录账号
2. 上传PDF并输入AIGC占比x
3. 系统解析文本并计算分级结果
4. 查看总览与标红报告
5. 下载PDF报告

## 结果说明
- **总览:** AIGC总占比x、高度疑似占比x*0.7、轻度疑似占比x*0.3
- **标红:** 与总览统计一致的疑似句子
