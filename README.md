## 🚀 快速开始

### 1️⃣ 环境准备
确保你已经安装了 `Go` 和 `Node.js` 环境。

### 2️⃣ 启动服务
```bash
# 启动后端 (Server)
cd server
go run cmd/api/main.go

# 启动前端 (Web)
cd web
npm run dev
```

### 3️⃣ 尽情享用
打开浏览器访问 `http://localhost:5173` (或终端提示的地址)，开启你的文检之旅吧！🎉
