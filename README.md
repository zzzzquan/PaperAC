## 🚀 快速开始

### 1️⃣ 环境准备
确保你已经安装了 `Node.js`（v18+）环境。

### 2️⃣ 安装依赖 & 启动
```bash
npm install
npm run dev
```

### 3️⃣ 尽情享用
打开浏览器访问 `http://localhost:5173`，开启你的文检之旅吧！🎉

### 📦 构建部署
```bash
npm run build
# 生成的 dist/ 目录可直接部署到任何静态托管服务
```
