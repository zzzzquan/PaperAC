# 高情商AIGC检测器 - 需求文档

## 1. 项目概述
公开Web服务，面向课程论文场景，用户上传PDF并获得AIGC检测报告。系统通过LLM API进行句子级判别，生成总览与标红原文报告，可下载PDF。

## 2. 目标与成功标准
- **检测目标:** 输出AIGC占比与可疑句子列表，结果不高于用户输入阈值
- **结果体验:** 语气友好、对用户体验“高情商”
- **合规与隐私:** 处理后不存储论文正文，仅保留必要元数据
- **可用性:** 公开服务稳定运行，支持域名访问

## 3. 目标用户与场景
- **用户:** 课程论文撰写者、教学助教、课程教师
- **场景:** 学生自检或教学辅助的检测

## 4. 核心流程
1. 用户注册/登录
2. 上传PDF并设置AIGC占比阈值（默认15%，最低5%）
3. 系统解析PDF、清洗正文、拆分句子
4. 调用LLM API判别句子AIGC概率
5. 根据阈值选择可疑句子并生成报告
6. 展示总览与标红原文，支持下载PDF报告
7. 任务完成后删除论文与中间文件

## 5. 功能需求
### 5.1 上传与任务管理
- 支持PDF上传
- 提供任务状态与进度
- 失败时提供明确提示

### 5.2 AIGC判别
- 支持Qwen或DeepSeek API
- 句子级判别并给出可解释字段
- **高情商约束:** 最终AIGC占比不高于用户阈值
- 阈值选择可选，默认15%，最低5%

### 5.3 报告生成
- **总览报告:** AIGC占比、可疑句子数量、阈值说明
- **原文报告:** 标红疑似句子
- **PDF下载:** 报告可导出PDF

### 5.4 账号与配额
- 支持邮箱/手机号注册登录
- 提供配额或计费机制（方案待定）

### 5.5 合规与免责声明
- 首页与报告页展示严格免责声明
- 仅供参考，不作为唯一依据

## 6. 非功能需求
- **隐私:** 不长期存储论文正文，处理完成后删除
- **安全:** 访问控制、API调用限流、滥用检测
- **性能:** 任务响应在可接受范围内（具体指标待定）
- **可扩展:** 支持多LLM后端切换

## 7. 部署与运维
- 部署平台: 阿里云
- 域名: 需要绑定公共域名
- 监控: 任务成功率、LLM调用失败率、响应时延

## 8. 风险与假设
- LLM判别准确率受提示词和上下文影响
- PDF解析质量影响结果
- 计费/配额与风控方案待确定
