# 🎓 情智文检 (PaperAC)

> **High EQ, High Quality!** ✨
> 这里的报告不仅仅是数字，更是懂你的“情商”担当！

👋 欢迎来到 **PaperAC**！我们是一款专注于课程论文场景的 **AIGC 检测神器**。不管你是被ddl追赶的大学生，还是严格把关的助教老师，我们都能帮你轻松搞定论文查重（含AI成分）！

## 🌟 核心亮点

- **🎯 精准分级**: 我们不玩虚的！检测结果按 **“高度疑似/轻度疑似”** 分级，比例固定为 **7:3**，科学又严谨！
- **🎛️ 自由掌控**: 你输入想要控制的 **AIGC 总占比 x**，不管是 5% 还是 15%，我们都帮你安排得明明白白！
- **🚀 极简操作**: 上传 PDF -> 输入占比 -> 搞定！就是这么丝滑～

## 🛠️ 功能大揭秘

| 功能 | 说明 |
| :--- | :--- |
| **📄 PDF 上传** | 拖拽上传，立即解析，大文件也不在话下 |
| **🔍 智能抽取** | 自动识别正文，把无关的封面、目录通通过滤掉 |
| **📊 占比控制** | 高低疑似合计 = 你输入的 x，不多不少，刚刚好 |
| **📋 标红报告** |生成的报告一目了然，哪里有问题指哪里 |
| **📥 结果归档** | 随时下载报告，记录你的每一次优化 |

## 🚀 快速开始

### 1️⃣ 环境准备
确保你已经安装了 `Go` 和 `Node.js` 环境。

### 2️⃣ 启动服务
```bash
# 启动后端 (Server)
cd server
go run cmd/api/main.go

# 启动前端 (Web)
cd web
npm run dev
```

### 3️⃣ 尽情享用
打开浏览器访问 `http://localhost:5173` (或终端提示的地址)，开启你的文检之旅吧！🎉
